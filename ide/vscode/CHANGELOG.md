# Change Log

All notable changes to the "golr" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- Syntax highlighting (TextMate grammar) and semantic highlighting that distinguishes symbol
  definitions from references and terminals from nonterminals.
- Go to Definition for GoLR symbols.
- Find All References for GoLR symbols.
- Rename Symbol across a symbol's definition and all of its references.
- Auto-completion of the symbols defined in the current file.
- Code formatting into the canonical GoLR layout.
- CodeLens showing the number of references above each symbol definition; clicking it opens
  the reference peek.
